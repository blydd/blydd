查看服务器DNS信息:

修改DNS服务器非常简单，只需要修改/etc/resolv.conf配置文件即可。那么修改DNS是否需要重启某些服务，答案是不需要。修改后会立即生效，不需做任何额外处理。

ping：测试网络命令

  -c count：ping的次数
-W timeout：超时时间，配合-c使用
-I ipaddress：指定用自己主机的IP去ping对方主机
-s size：每次ping发出的数据包大小，最大值65507
-f：竭尽自己主机的能力发出数据包

常见错误

1.yum安装时报cannot find a valid baseurl for repo

出现这个问题是因为yum在安装包的过程中，虽然已经联网，但是没法解析远程包管理库对应的域名，所以我们只需要在网络配置中添加上DNS对应的ip地址即可。

​ 1.查看网卡名:ip addr

​ 2.修改网络配置文件vi /etc/sysconfig/network-scripts/ifcfg-ens33

​ 3.重启网络ifup ens33,成功

2.ifconfig命令报-bash: ifconfig: command not found

安装网络工具:yum install -y net-tools

3.CentOS下ifconfig命令不显示ip

1.查看网卡名:ip addr,我的显示ens33

2.修改配置文件:/etc/sysconfig/network-scripts/ifcfg-ens33,ONBOOT改为yes

3.启动网卡ifup ens33即可.
