端口

netstat -an #查看本机启用的端口

netstat -tuln #查看本机开了哪些端口和正在被远程调用的调用方ipnetstat -tuln 查看本机开了哪些端口和正在被远程调用的调用方ip

telnet [域名或ip] [端口] #远程管理与端口探测命令
eg:telnet 192.168.0.252 80 #探测服务器80端口,成功的话没有提示,进入空白页,退出摁ctrl+],退回到telnet界面,输入quit命令退出

查看服务器DNS信息:

修改DNS服务器非常简单，只需要修改/etc/resolv.conf配置文件即可。那么修改DNS是否需要重启某些服务，答案是不需要。修改后会立即生效，不需做任何额外处理。

ping：测试网络命令

  -c count：ping的次数
-W timeout：超时时间，配合-c使用
-I ipaddress：指定用自己主机的IP去ping对方主机
-s size：每次ping发出的数据包大小，最大值65507
-f：竭尽自己主机的能力发出数据包

查看DNS服务器地址:

命令行登陆linux服务器:

ssh -p22 用户名@ip

wget http://soft.vpser.net/Inmp/Inmp1.1-full.tar.gz #下载命令

crl软件按:alt p

浏览文件内容

  • cat 文件名

    适合查看数据量小的文件,全部展示.

  • more 文件名

    分页显示文件内容

    空格:下一页 回车:下一行 ctrl+b:上一屏 q:退出

  • less 文件名

    分页查看文件内容.操作同more.

    标注行号:less -mN 文件名

  • tail -20 文件名

    快速查看文件最后20行内容.

  • Vim 文件名

    输入 : set nu,会显示行号.

grep和管道

| 是管道符号,可连接不同命令

grep是正则表达式,用于字符串搜索.格式:grep 字符串 搜索的文件,eg:ll | grep aaa
