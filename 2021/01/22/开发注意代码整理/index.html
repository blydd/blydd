拼接in字段

String mergeString = list.stream().map(demo -> “‘“ + demo.getUserCode() + “‘“).distinct().collect(Collectors.joining(“,”));

stream操作集合时注意判空,否则遇空会报错

usercodess = users.stream().filter(map -> StringUtils.isNotBlank((String) map.get("userCode"))).collect(Collectors.groupingBy(map -> (String) map.get("userCode")));

stream的group操作1:value是集合

Map<String, List<Map<String, Object>>> usercodess = users.stream().filter(map -> StringUtils.isNotBlank((String) map.get("userCode"))).collect(Collectors.groupingBy(map -> (String) map.get("userCode")));

stream的group操作2:value是集合中对象某字段的和

Map<String, Integer> collect1 = resultList.stream().filter(item -> StringUtils.isNotBlank(item.getGroupCode()))
	.collect(Collectors.groupingBy(ComplCountVO::getGroupCode, Collectors.summingInt(ComplCountVO::getComplAmount)));

mybatis参数条件集合

if条件判断时需注意，如果字段类型是Integer，则不能使用,因为Integer无法和空字符串做比较。

<if test="roomCodeList!= null and roomCodeList.size>0">
	and b.room_code in
	<foreach collection="roomCodeList" index="index" open="(" close=")" separator="," item="item" >
		#{item}
	</foreach>
</if>

GET方式传参为对象或map时加注解 @SpringQueryMap
POST方式传参记得加注解 @RequestBody

写swagger代码

  • 对象入参
    @ApiOperation(value = "方法功能")
       @ApiImplicitParams({
               @ApiImplicitParam(name = "paramDTO", required = true, dataType = "ParamDTO", value = "字段含义", paramType = "body"),
       })
    @Data
    @Accessors(chain = true)
    @ApiModel(description = "参数对象说明")
    public class FeeOrderDTO {

        @ApiModelProperty(value = "字段名称", example = "传参示例", required = true)
        @NotBlank(message = "业主编码不能为空")
        private String  ownerCode;
    }
  • 零散入参

    })

import com.alibaba.excel.EasyExcel;

   public void exportExcel(HttpServletResponse response, ParamVO paramVO) throws IOException {
       Assert.isBlank(paramVO.getUserCode(), "用户编码不能为空");
	SimpleDateFormat df = new SimpleDateFormat("yyyyMMddHHmmssSSS");
       // 组装导出数据
       List<ExportData> recordExportData = testDaoGetList(paramVO);
       if (CollectionUtils.isEmpty(recordExportData)) {
           throw new ServiceException("不存在要导出的数据");
       } else {
           response.setContentType("application/vnd.ms-excel");
           response.setCharacterEncoding("utf-8");
           String tempName = "文件名" + df.format(new Date());
           log.info("导出文件名是:" + tempName);
           String fileName = URLEncoder.encode(tempName, "UTF-8").replaceAll("\\+", "%20");
           response.setHeader("Content-disposition", "attachment;filename*=utf-8''" + fileName + ".xlsx");
           // 获取要导出的模板数据
           EasyExcel.write(response.getOutputStream(), ExportData.class).sheet("template").doWrite(recordExportData);
       }
   }

import com.alibaba.excel.annotation.ExcelProperty;
import com.alibaba.excel.annotation.write.style.ColumnWidth;
import lombok.Data;

import java.math.BigDecimal;

/**
 * 功能描述：导出模板

//单元格宽度
@ColumnWidth(value = 20)
public class ExportData {
    @ExcelProperty(value = "列名1")
	private BigDecimal money;
    @ExcelProperty(value = "列名2")
	private String name;
}
